# Análisis de Datos con Python (IBM)

Este directorio contiene los laboratorios y ejercicios prácticos del curso **"Data Analysis with Python"** de la certificación profesional de IBM.

## Contenido

### [Módulo 1: Introducción al Análisis de Datos]()
* **Carga de datos:** Importación de datasets (CSV) utilizando Pandas.
* **Limpieza de datos:** Manejo de cabeceras, identificación de valores nulos y corrección de tipos de datos.
* **Análisis Exploratorio:** Uso de funciones estadísticas básicas (`describe`, `info`) para obtener insights iniciales.
