# Data Analysis with Python (IBM)

Este directorio contiene los laboratorios y ejercicios prácticos del curso **"Data Analysis with Python"** de la certificación profesional de IBM.

## Contenido

### Módulo 1: Introduction to Data Analysis
* **Carga de datos:** Importación de datasets (CSV) utilizando Pandas.
* **Limpieza de datos:** Manejo de cabeceras (headers), identificación y tratamiento de valores nulos (NaN) y corrección de tipos de datos.
* **Análisis Exploratorio:** Uso de funciones estadísticas básicas (`describe`, `info`) para obtener insights iniciales del dataset de automóviles.
